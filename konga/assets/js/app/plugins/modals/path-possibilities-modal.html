The main idea is to support 3 types of dynamic content:
  • ? match any one path element
  • ?? match zero or more path elements
  • {regexp} - match single path element against PCRE

Priority:
  • Exact match
  • Regexp match
  • ?
  • ??

Note: slash is required before multiple wildcards placeholder.

Examples: Assume that below all path is register in one plugin
Register Path Allow path Deny path
{{path.path}}
  • {{allow}}
  • {{deny}}
